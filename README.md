# Tic-Tac-Toe Game

A simple Tic-Tac-Toe game built with React to practice React fundamentals.

## Features

- 🎮 Classic Tic-Tac-Toe gameplay
- ⚛️ Built with React
- 🔄 Interactive UI
- 🧩 Component-based architecture
- 📱 Responsive layout

## Technologies

- React
- JavaScript (ES6+)
- CSS
- Vite

## Getting Started

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

## Learning Objectives

This project was built to practice:

- React Components
- Props
- State Management
- Event Handling
- Conditional Rendering
- React Hooks
